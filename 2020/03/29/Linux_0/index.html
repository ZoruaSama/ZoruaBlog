Linux

大多数的00后应该和我一样，第一次接触PC操作系统应该是Windows，那就先来说说与Windows的异同。

  • 在Windows 7以前版本的Windows是单用户操作系统，而Linux的设计初衷就是多用户操作系统。
  • 在Linux下，没有驱动器盘符，即没有你在Windows中看到的C:\，D:\，E:\等等，Linux中仅有/，称之为根目录。
  • 在Windows下，普通用户基本都是在纯图形界面下操作使用；而Linux兼具图形界面操作（桌面环境的发行版）和终端操作。几乎可以只用键盘完成一切操作。
  • 世界上绝大多数服务器都正在运行着Linux。

当然还有很多不同的地方，我这里就只稍微例举一下下。

以下是百度百科摘取的一段关于Linux的介绍。

Linux，全称GNU/Linux，是一套免费使用和自由传播的类UNIX操作系统，其内核由林纳斯·本纳第克特·托瓦兹于1991年第一次释出，它主要受到Minix和Unix思想的启发，是一个基于POSIX和Unix的多用户、多任务、支持多线程和多CPU的操作系统。它能运行主要的Unix工具软件、应用程序和网络协议。它支持32位和64位硬件。Linux继承了Unix以网络为核心的设计思想，是一个性能稳定的多用户网络操作系统。Linux有上百种不同的发行版，如基于社区开发的debian、archlinux，和基于商业开发的Red Hat Enterprise Linux、SUSE、oracle linux等。

比较著名的发行版本有Debian、RedHat、Centos、Slackware、clearlinux、ubuntu等等。无论何种版本的 Linux 发行版，桌面、应用是Linux的外衣，文件组织、目录结构才是Linux的内心。

我使用的是ubuntu-16.04.6，可以在其官网获取相关镜像。Ubuntu官网
如果下载速度可能比较缓慢，可以在清华大学开源软件镜像站或其他镜像站进行下载。
由于技术所限，我并没有在我的计算机上安装双系统，是用虚拟机软件进行模拟的。
常用的虚拟机软件有VirtualBox VMware Workstation Virtual PC等等。请自行在互联网上获取。

Windows的Pro版本好像自带一个Hyper-V。

文件和目录

上面说到Linux中仅有根目录/，下面是根目录下，一些常用的目录的简单描述。

目录 描述
/ 有且仅有一个的根目录，所有的文件都存放在这里。
/bin、/usr/bin 存放着可执行二进制文件的目录，如常用的命令 ls、cat 等等。
/home 家目录，用户的家目录都存放在该目录下。
/etc 系统配置文件存放的目录。
/dev 存放在系统下的设备文件，访问该目录的文件相当于访问某个设备。

另外还有些/root /proc等就暂且不提辽。
着重提一下家目录，/home下存放着各个用户的家目录，例如一个Ubuntu系统上有Zorua Zoroark两个用户，则/home下就会有两个家目录：/home/Zorua /home/Zoroark。每个用户的家目录大概可以类比为Windows的C:/Users/Zorua这一文件夹吧。
在Linux中，当前用户的家目录路径可以用~来表示，例如当前用户名为Zorua，则~等价于/home/Zorua。
每个用户的家目录下，也有很多的文件夹。

  • 在Linux的目录系统中，文件或文件名前面加一个.，就是隐藏文件的意思，隐藏文件默认不能直接在图形界面或终端的ls命令下看到。
  • 有两个特殊的隐藏目录，之所以说是隐藏，是因为只能用ls -a才能看到。一个是..，一个是.。
    ..表示这个目录的上一级目录，.表示当前目录。这两个目录都是相对路径。
    例如对于目录/home/Zorua，它的..就意味着/home，.就意味着/home/Zorua。
    这两个目录在任何目录都有的（是的，包括根目录，根目录的 ..还是根目录）。
